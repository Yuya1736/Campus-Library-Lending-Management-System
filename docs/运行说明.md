# Campus Library Lending Management System

## 1. 环境
- .NET SDK 8+
- Visual Studio 2022
- MySQL 8.x（本地）
- Navicat

## 2. 数据库初始化（你需要手动操作）
1. 打开 Navicat，连接你的本地 MySQL。
2. 依次执行：
   - `database/schema.sql`
   - `database/seed.sql`
3. 确认数据库名为 `campus_library`。

## 3. 配置连接字符串（你需要手动操作）
编辑文件：`CampusLibrary.WinForms/Config/AppConfig.cs`

把 `ConnectionString` 改成你本机配置，例如：
`Server=127.0.0.1;Port=3306;Database=campus_library;Uid=root;Pwd=你的密码;CharSet=utf8mb4;AllowUserVariables=True`

## 4. 运行项目
```bash
dotnet build CampusLibrary.slnx
dotnet run --project CampusLibrary.WinForms/CampusLibrary.WinForms.csproj
```

## 5. 默认登录账号
- 管理员：`admin` / `123456`
- 操作员：`operator` / `operator123`

## 6. 已实现模块
- 登录与权限控制（管理员/操作员）
- 图书基础管理（增删改查）
- 借书与还书（库存联动、超期计算）
- 超期名单刷新
- 统计分析（热门图书、分类借阅率）
- 用户管理（仅管理员）
